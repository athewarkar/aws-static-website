# AWS Static Website Project

## Overview
In this project, I built and deployed a simple static website using Amazon S3. It helped me understand how cloud storage can be used to host real websites and how to configure access properly.

## What I Did
- Built a basic HTML website
- Uploaded the files to an S3 bucket
- Set up static website hosting
- Configured permissions so the site is publicly accessible

## Live Website
[Live Website](http://aditi-cloud-portfolio-123.s3-website-us-east-1.amazonaws.com/)

## Tools Used
- AWS S3
- HTML

## Skills Demonstrated
- Deploying applications on AWS
- Working with S3 for hosting
- Understanding public access and permissions
- Basic cloud fundamentals
